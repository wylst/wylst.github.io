We are a software and design studio, and we make simple, open source utilities for the web.

  • http.js
    Tiny client-side JavaScript library to simplify AJAX/HTTP requests.
  • atom-minifier & brackets-minifier
    Minify JavaScript and CSS in Atom and Brackets with UglifyJS and the YUI compressor.
  • dolly
    Simple GitHub repository cloning with nothing but PHP.
  • fllat & prequel
    A flat-file, JSON-based database system written in PHP, and a PHP library for SQL-like operations on arrays.
  • pretty-date
    John Resig's JavaScript and jQuery plugin to humanize ISO dates, modified to work.
